html and css page setup

Concept

Description

Syntax
!DOCTYPE
  • Is an instruction and must be the first line of code in a HTML document
  • Lets browsers know we are using HTML
  • Anything between these tabs will be interpreted as HTML code
  • Provides a) type of document and b) HTML version to expect
  • Does not require closing tag
  1. <!DOCTYPE html>
  2. <html>
  3. </html>
head
  • Goes above <body>
  • Contains metadata for a webpage
  • <title>is always located inside the <head>. Displays the specified title in the browser's tab.
<head>
Linking CSS file
  • The <link> element must be placed withing the <head>
  • It is a self closing tag
  • <link href= 'url' rel= 'stylesheet'>
  • <link href= 'relative path' rel= 'stylesheet'>
The body Only content inside the body tags can be displayed to the screen
  1. <body>
  2. </body>
